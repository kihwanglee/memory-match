# 기억력 카드 매칭 게임 아이디어 브레인스토밍

| 구분 | 내용 |
|------|------|
| 개요 | 뒤집힌 카드들 중 동일한 그림을 가진 카드 두 장을 맞추는 기억력 향상 게임 |
| 목적 | 기억력 강화, 집중력 향상, 남녀노소 누구나 즐길 수 있는 간단한 두뇌 훈련 제공 |
| 사용자 | 어린이, 청소년, 성인, 두뇌 트레이닝 필요 이용자 모두 대상 |
| 핵심 기능 | - 카드 뒤집기 & 매칭<br>- 난이도 설정(카드 수 증가)<br>- 시간/턴 제한 모드<br>- 점수 및 기록 관리<br>- 시각/효과음 피드백<br>- 모바일/웹 반응형 UI |
| 제약 조건 | - 카드 이미지 라이선스 확보<br>- 제한된 화면 크기 고려<br>- 단순하지만 중독성 있는 UX 필요<br>- 난이도 지나친 상승 방지 |

---

### 📌 추가 구현 아이디어
- 테마 선택: 동물, 음식, 국기, 캐릭터 등
- 멀티플레이: 턴 교대 대전 모드
- 학습 요소 추가: 카드 뒷면 정보 제공 (예: 동물 이름/국가 이름)
